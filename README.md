### Приложение для прогноза погоды  
  
#### Стек  
- Retrofit для загрузки данных
- Kotlin serialization для парсинга
- Kotlin coroutines и Flow
- Room
- Coil для загрузки изображений
  
#### Экраны
- Текущий прогноз погоды
- Прогноз по дням
- Поиск города
- Избранные города
- Настройки единиц измерений
